# Password Manager

### About

This is a monorepo for a full-stack application used to save and manage passwords. You can check out the live app by clicking the URL below.

https://client-app-production-28f3.up.railway.app/

### Built With

This app is built using the following languages and tools.

- Backend

  - [Golang](https://go.dev/doc/install)
  - [PostgreSQL](https://www.postgresql.org/download/)
  - [Gorm](https://gorm.io/)
  - [Gin](https://gin-gonic.com/)

- Frontend
  - [TypeScript](https://www.typescriptlang.org/)
  - [Next JS](https://nextjs.org/docs)
  - [Tailwind](https://tailwindcss.com/)
